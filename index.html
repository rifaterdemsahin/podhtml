Key Concepts

    About OpenShift

    OpenShift is a hybrid cloud, enterprise Kubernetes platform that helps organizations develop, deploy, and manage applications in containers. Some of the key concepts include:

    • Containers: Portable units that bundle applications with their dependencies.
    • Kubernetes: An orchestration system for automating deployment and scaling.
    • CI/CD: Continuous Integration/Continuous Deployment pipelines for automated development workflows.
    • Operator Framework: Tools to manage the lifecycle of containerized applications.
    • Routes and Services: Expose and manage network traffic to applications.
